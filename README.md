# chatgpt-master

#### 介绍
Java+gpt-3.5-turbo聊天应用,后端代码
使用前后端分离，前端用vue代码见另一个仓库

#### 软件架构
springboot


#### 安装教程

1.  后端部署时需要在配置文件application.yaml中将openai的apiKey替换会你自己的apiKey
